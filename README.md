Unsemantic for QUIQQER
========

Integrates the Unsemantic CSS Framework into QUIQQER.
Makes using Unsemantic in your QUIQQER projects easy.


Package name:

    quiqqer/unsemantic


Features
--------

- CSS Unsemantic Framework


Installation
------------

The package name is: `quiqqer/unsemantic`
